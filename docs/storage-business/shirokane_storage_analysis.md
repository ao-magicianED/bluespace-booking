# ブルースペース白金高輪の屋内ストレージ（無人運営）業態変更に関する分析・戦略提言

## 1. エグゼクティブサマリー

ブルースペース白金高輪（15㎡・家賃7万円）を屋内ストレージの無人運営に業態変更し、11〜15万円の売上を目指すという構想について、**「十分に勝ち筋はあり、実現可能性・収益性ともに高い」**と結論付けられます。

最大の強みは「15㎡（約9畳）を1室丸ごと貸し出す」というニッチなポジショニングです。港区白金高輪エリアの競合（キュラーズなど）は小規模（1〜2畳）が中心であり、大型サイズ（3畳以上）は軒並み満室で強い需要があります。法規制の面でも、200㎡未満であるため用途変更申請が不要であり、初期投資を抑えたスピーディな立ち上げが可能です。

一方で、1室貸し切りモデルは「稼働率が0%か100%の二極化」となるため、最初の1社（または1人）を獲得するまでの空室期間をいかに乗り切るかが成功の鍵となります。

## 2. 白金高輪エリアの競合・需要環境

白金高輪エリアは、高級住宅街とビジネスエリアが混在しており、収納ニーズが非常に高い地域です。

### 競合の状況と料金相場
最大の直接競合となる「キュラーズ白金高輪店」（徒歩6分）の料金表を分析した結果、明確な市場の隙間が確認できました[1]。
同店では、0.5畳〜1.6畳の小型・中型サイズも一部満室ですが、2.5畳、3.3畳、5.0畳といった**大型サイズがすべて満室**となっています。

| :--- | :--- | :--- | :--- |
| キュラーズ白金高輪 3.3畳 | 約5.5㎡ | 107,140円 | 約32,467円/畳 |
| キュラーズ白金高輪 5.0畳 | 約8.3㎡ | 159,720円 | 約31,944円/畳 |

*出典: キュラーズ公式サイト料金表、収納ピット市場データより算出*

### 15㎡（約9畳）の価格妥当性
あおさんが想定されている「月額11〜15万円」という価格設定は、市場相場に照らして**非常に競争力があり妥当**です。
15㎡を約9畳と換算した場合、キュラーズの定価ベースでは約28万円相当の価値になります。一般的な港区の相場（畳単価約1.5万円）で計算しても約13.5万円となります。したがって、11〜15万円という設定は、借り手にとって「相場並み〜割安」に映る絶妙な価格帯です。

## 3. 法規制と無人運営の実現方法

### 法規制のクリア（用途変更不要）
既存のレンタルスペースからトランクルームへの業態変更において、最も大きなハードルとなるのが建築基準法上の「用途変更」です。しかし、令和元年6月の法改正により、**200㎡未満の施設は用途変更申請が不要**となりました[2]。
本物件は15㎡であるため、この手続きは不要です。また、自社の物品を預かるのではなく「スペースを貸す（場所貸し）」形式のレンタル収納スペースとして運営すれば、倉庫業法の登録も不要となります[3]。

### 無人運営システムの導入
完全無人運営を実現するためには、スマートロックと防犯カメラの導入が必須です。
クラウド管理型のスマートロック（例：RemoteLOCKやSADIOT ROOMなど）を導入することで、遠隔からの鍵の発行・管理が可能になります[4]。

| 設備・システム | 初期費用目安 | 月額費用目安 | 備考 |
| :--- | :--- | :--- | :--- |
| スマートロック本体＋工事 | 70,000〜110,000円 | 1,650円 | 機種により変動 |
| Wi-Fi通信環境 | 既存利用なら0円 | 4,000円 | 遠隔管理に必須 |
| クラウド録画型防犯カメラ | 20,000〜30,000円 | 1,500円 | トラブル防止に必須 |
| **合計** | **90,000〜140,000円** | **約7,150円** | 最小構成の場合 |

*出典: 各社スマートロック提供企業の料金表に基づく試算*

## 4. 収益性シミュレーション

1室丸ごと貸し出しモデルの収益性を試算します。初期投資は内装工事を最小限（現状の壁・床を活かす）とし、スマートロック、防犯カメラ、除湿機程度の導入にとどめた「最小構成」を前提とします。

**初期投資概算:** 約15〜25万円（システム導入、ポータルサイト登録料、備品等）

**月額ランニングコスト:**
- 家賃: 70,000円
- システム・通信費: 約7,150円
- 水道光熱費（除湿機等）: 約5,000円
- 保険料・その他: 約3,000円
- **合計固定費: 約85,150円**

**収益シミュレーション:**
- **売上 110,000円の場合:** 月間利益 24,850円（年間約30万円）
- **売上 130,000円の場合:** 月間利益 44,850円（年間約53万円）
- **売上 150,000円の場合:** 月間利益 64,850円（年間約77万円）

初期投資が15〜25万円と少額であるため、契約者が決まれば**半年〜1年以内で初期投資を回収可能**という非常に優秀な投資効率となります。

## 5. 勝ち筋と戦略提言

この事業モデルを成功させるための具体的な「勝ち筋」を3点提言します。

### 戦略1: 法人・EC事業者向けの「大型個室」としてのポジショニング
15㎡を細かくパーティションで区切るのではなく、1室丸ごと貸し出す戦略は正解です。パーティション工事費（数十万円）を節約できるだけでなく、周辺で枯渇している「大型サイズ需要」を独占できます。
ターゲットは個人ではなく、**周辺の法人（書類や備品保管）や小規模EC事業者（在庫保管・発送拠点）**に絞るべきです。電源やWi-Fiが使える点をアピールすれば、単なる倉庫ではなく「軽作業もできるミニ倉庫」として付加価値が高まります。

### 戦略2: 空室期間を乗り切る「ハイブリッド募集」
1室貸しの最大のリスクは「契約者が決まるまでの売上がゼロ」になることです。
これを回避するため、現在のレンタルスペースとしての募集（時間貸し）を継続しながら、同時にトランクルーム検索ポータル（eトランクなど）や法人向け不動産サイトで「月極の大型ストレージ」として募集をかける**ハイブリッド期間**を設けることを推奨します。月極の契約者が決まった時点で、時間貸しを停止して業態を完全移行させます。

### 戦略3: 初期投資を極限まで抑えたリーンスタート
内装工事は行わず、現状のスペースをそのまま「大型個室トランクルーム」として提供します。必要なのは、確実なセキュリティ（スマートロックとカメラ）と、保管環境（除湿機）のみです。利用規約をしっかり整備し、トラブル時の責任分解点を明確にしておくことで、管理の手間を最小限に抑えられます。

## 6. 結論

ブルースペース白金高輪を15㎡・家賃7万円の条件で、11〜15万円の無人ストレージへ転換する計画は、**市場の需給ギャップ（大型サイズの不足）を突く合理的な戦略**であり、十分に勝ち筋があります。

パーティションで細分化する一般的なトランクルームとは異なり、1室貸し切りモデルは「入居者が決まるか否か」の0か100かの勝負になりますが、初期投資が極めて低いためリスクは限定的です。法人やEC事業者をターゲットに「電源・Wi-Fi付きの個室ミニ倉庫」として訴求することで、安定した長期契約を獲得できる可能性が高いと評価します。
