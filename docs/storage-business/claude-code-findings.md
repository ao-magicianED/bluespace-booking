## 3. 月極ストレージLPのCVRベストプラクティス

### ファーストビューに置くべき要素（優先順）

1. **ターゲットを直接呼ぶヘッドライン**（例: 「EC事業者・法人向け」と明示）
2. **主要ベネフィット3つ以内**（「24h入退室」「即日利用可」「月額178,000円〜」）
3. **メインCTA1個のみ**（「内覧申込み（無料）」か「今すぐ問い合わせ」）
4. **施設の実写真**（スタジオ写真でなく「実際の空間」）
5. **希少性バナー**（「残り1室」を赤字・目立つ位置に）

### CTA配置の鉄則

- CTAは「1〜2個に絞る」（3個以上は迷いを生み離脱増）
- ファーストビュー内に1個 + 各セクション末尾に繰り返し
- ボタン文言は「行動が具体的」なもの（「問い合わせる」より「内覧を申込む（無料）」）
- ボタンカラーは背景と高コントラスト（例: 濃紺背景なら白・オレンジ）

### 信頼感を生む要素

| 要素 | 実装方法 |
|------|---------|
| 設備写真（実写） | 室内・スマートロック・カメラ・除湿機の写真4枚以上 |
| 運営会社情報 | ブルーステージ合同会社の法人情報（所在地・設立年・代表者） |
| セキュリティ説明 | 24hカメラ録画・スマートロック（ICカード不要） の具体的説明 |
| 入居実績/利用者声 | EC事業者・店舗オーナーの1〜2件の声（テキストのみでも可） |
| 契約形式の安心感 | 「倉庫業登録なし」「スペース貸し契約」「物品預かり業に該当しない」の明記 |
| アクセス地図 | Googleマップ埋め込み（白金高輪駅からの経路） |

### モバイル最適化のポイント

- CTAボタンは **最小44px高さ**、フィンガータップを想定
- 電話番号はタップでコール（`<a href="tel:...">`）
- ファーストビューは **スクロール不要でCTAが見える**（モバイル画面 667px以上を基準）
- フォームは **入力項目3個以内**（名前・電話番号・メッセージ or 希望日）
- 画像はWebP・遅延ロード（LCP 2.5秒以内を目指す）

## 5. 実装上の推奨事項（総合）

### LP構成（推奨セクション順）

1. **ファーストビュー**: ヘッドライン + 施設実写 + 「残り1室」バナー + CTA
2. **課題提起**: 「こんな悩みありませんか？」（在庫問題・バックヤード不足）
3. **解決策**: 15㎡まるごと専有の特徴 + 主要設備リスト
4. **設備写真ギャラリー**: 室内・スマートロック・カメラ・間取り図
5. **料金**: 月額178,000円（税込）/ 6ヶ月以上158,000円 + 費用内訳の透明性
6. **アクセス**: Googleマップ埋め込み + 白金高輪駅からの経路説明
7. **FAQ**: 「倉庫業との違いは？」「法人請求書払いは？」等を含める
8. **運営会社**: ブルーステージ合同会社の会社情報（信頼の根拠）
9. **CTA（クロージング）**: 内覧申込みフォーム（3項目以内）

### Manus調査への追加・修正情報

- 競合ポータル（japantrunkroom.com、awele.co.jp、e-trunk.jp）への掲載がSEO上有効（未言及なら追加推奨）
- スペラボの法人LPは「請求書払い対応」「電源付き」「即日契約」を前面に出している → ブルーストレージでも請求書払い対応の明記を検討
- 構造化データは `SelfStorage` 型（`RealEstateListing` は不適切）
- 「限定1室」の希少性訴求はLP全体を通じて反復すること（ファーストビュー・料金セクション・CTA前後で計3回以上）
- フォーム入力項目は3個以内に絞る（名前・連絡先・希望日時 or コメント）
- モバイルCTAは電話タップリンクを必ず用意（問い合わせ電話番号のクリック率は高い）
